title: "Autocomplete"
description: "Un componente de autocompletado inteligente con soporte para grupos, animaciones y diseño optimizado."
---

## Instalación

```bash
# Instalar dependencias necesarias
npm install framer-motion lucide-react

# Instalar el componente
npx @stofli/ui add autocomplete
```

## Uso

El componente Autocomplete proporciona una experiencia de búsqueda y selección intuitiva con animaciones fluidas.

```tsx
import { Autocomplete } from "@stofli/ui";

export default function App() {
  return (
    <Autocomplete
      options={[
        { value: "react", label: "React" },
        { value: "vue", label: "Vue" },
        { value: "angular", label: "Angular" },
      ]}
      placeholder="Selecciona un framework..."
    />
  );
}
```

## Props

| Prop | Tipo | Default | Descripción |
| --- | --- | --- | --- |
| `options` | `Array<{ value: string, label: string, group?: string, disabled?: boolean }>` | `[]` | Las opciones disponibles en el dropdown. Cada opción puede ser deshabilitada individualmente con la propiedad `disabled: true` |
| `value` | `string` | - | El valor seleccionado actualmente |
| `onChange` | `(value: string) => void` | - | Función llamada cuando cambia la selección |
| `placeholder` | `string` | `"Selecciona una opción..."` | Texto mostrado cuando no hay selección |
| `title` | `string` | - | Título que se muestra sobre el campo |
| `required` | `boolean` | `false` | Indica si el campo es obligatorio, mostrando un asterisco rojo junto al título |
| `variant` | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"` | Variante de color del componente |
| `className` | `string` | `""` | Clases adicionales para el componente |
| `disabled` | `boolean` | `false` | Desactiva el componente |
