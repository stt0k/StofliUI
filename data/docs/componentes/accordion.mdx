---
title: "Accordion"
description: "Un componente de acordeón que permite mostrar y ocultar contenido de manera interactiva."
---

## Instalación

```bash
# Instalar dependencias necesarias
npm install framer-motion lucide-react

# Instalar el componente
npx @stofli/ui add accordion
```

## Uso

El componente Accordion acepta un array de items con título y contenido, y opcionalmente puede configurarse para permitir múltiples secciones abiertas.

```tsx
import { Accordion } from "@stofli/ui";

export default function App() {
  return (
    <Accordion
      items={[
        {
          title: "¿Qué es un Accordion?",
          content:
            "Un accordion es un componente que permite mostrar y ocultar contenido...",
        },
        {
          title: "¿Cuándo debería usarlo?",
          content:
            "Los accordions son útiles cuando necesitas presentar mucha información...",
        },
      ]}
    />
  );
}
```

## Props

| Prop               | Tipo                                      | Default | Descripción                                           |
| ------------------ | ----------------------------------------- | ------- | ----------------------------------------------------- |
| `items`            | `{ title: string, content: ReactNode }[]` | -       | Array de elementos con título y contenido             |
| `defaultOpen`      | `number`                                  | `-1`    | Índice del item abierto por defecto (-1 para ninguno) |
| `allowMultiple`    | `boolean`                                 | `false` | Permite abrir múltiples secciones a la vez            |
| `className`        | `string`                                  | -       | Clases para el contenedor principal                   |
| `itemClassName`    | `string`                                  | -       | Clases para cada elemento del acordeón                |
| `buttonClassName`  | `string`                                  | -       | Clases para los botones del acordeón                  |
| `titleClassName`   | `string`                                  | -       | Clases para los títulos del acordeón                  |
| `contentClassName` | `string`                                  | -       | Clases para el contenido del acordeón                 |
| `iconClassName`    | `string`                                  | -       | Clases para los iconos del acordeón                   |
