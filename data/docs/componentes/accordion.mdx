    <HeadingAnchor id="props">Props</HeadingAnchor>

| Prop               | Tipo                                      | Default | Descripción                                           |
| ------------------ | ----------------------------------------- | ------- | ----------------------------------------------------- |
| `items`            | `{ title: string, content: ReactNode }[]` | -       | Array de elementos con título y contenido             |
| `defaultOpen`      | `number`                                  | `-1`    | Índice del item abierto por defecto (-1 para ninguno) |
| `allowMultiple`    | `boolean`                                 | `false` | Permite abrir múltiples secciones a la vez            |
| `className`        | `string`                                  | -       | Clases para el contenedor principal                   |
| `itemClassName`    | `string`                                  | -       | Clases para cada elemento del acordeón                |
| `buttonClassName`  | `string`                                  | -       | Clases para los botones del acordeón                  |
| `titleClassName`   | `string`                                  | -       | Clases para los títulos del acordeón                  |
| `contentClassName` | `string`                                  | -       | Clases para el contenido del acordeón                 |
| `iconClassName`    | `string`                                  | -       | Clases para los iconos del acordeón                   |
