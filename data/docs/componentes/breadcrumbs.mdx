---
title: "Breadcrumbs"
description: "Un componente de breadcrumbs para mostrar la navegación jerárquica de forma clara y accesible."
---

## Instalación

```bash
# Instalar dependencias necesarias
npm install lucide-react

# Instalar el componente
npx @stofli/ui add breadcrumbs
```

## Uso

```tsx
import { Breadcrumbs } from "@stofli/ui";

export default function App() {
  return (
    <Breadcrumbs
      items={[
        { label: "Inicio" },
        { label: "Componentes" },
        { label: "Breadcrumbs" },
      ]}
    />
  );
}
```

## Props

| Prop        | Tipo                                                                          | Default            | Descripción                             |
| ----------- | ----------------------------------------------------------------------------- | ------------------ | --------------------------------------- |
| `items`     | `{ label: string; href?: string; icon?: React.ReactNode }[]`                  | `[]`               | Elementos de la ruta de navegación      |
| `separator` | `React.ReactNode`                                                             | `<ChevronRight />` | Icono o texto separador entre elementos |
| `homeIcon`  | `boolean`                                                                     | `false`            | Muestra un icono de inicio al principio |
| `clickable` | `boolean`                                                                     | `false`            | Determina si los items son enlaces      |
| `variant`   | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"`        | Variante de estilo                      |
| `size`      | `"sm" \| "md" \| "lg"`                                                        | `"md"`             | Tamaño del componente                   |
| `className` | `string`                                                                      | -                  | Clases adicionales para el componente   |
