---
title: "Dropdown"
description: "Un componente de menú desplegable con animaciones y múltiples opciones de personalización."
---

## Instalación

```bash
# Instalar dependencias necesarias
npm install framer-motion lucide-react

# Instalar el componente
npx @stofli/ui add dropdown
```

## Uso

```tsx
import { Dropdown } from "@stofli/ui";

export default function App() {
  return (
    <Dropdown
      items={[
        { label: "Opción 1", value: "1" },
        { label: "Opción 2", value: "2" },
        { label: "Opción 3", value: "3" },
      ]}
      onChange={(value) => console.log("Valor seleccionado:", value)}
    />
  );
}
```

## Props

| Prop               | Tipo                                                                          | Default          | Descripción                                          |
| ------------------ | ----------------------------------------------------------------------------- | ---------------- | ---------------------------------------------------- |
| `items`            | `Array<DropdownItem>`                                                         | -                | Lista de elementos del dropdown                      |
| `placeholder`      | `string`                                                                      | "Seleccionar..." | Texto a mostrar cuando no hay selección              |
| `variant`          | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"`      | Variante de estilo                                   |
| `size`             | `"sm" \| "md" \| "lg"`                                                        | `"md"`           | Tamaño del dropdown                                  |
| `radius`           | `"none" \| "sm" \| "md" \| "full"`                                            | `"md"`           | Radio de las esquinas                                |
| `disabled`         | `boolean`                                                                     | `false`          | Deshabilita todo el dropdown                         |
| `fullWidth`        | `boolean`                                                                     | `false`          | Hace que el dropdown ocupe todo el ancho disponible  |
| `width`            | `string`                                                                      | -                | Define un ancho específico (ej: "200px", "15rem")    |
| `placement`        | `"bottom" \| "top" \| "left" \| "right"`                                      | `"bottom"`       | Posición del menú desplegable                        |
| `onChange`         | `(value: string) => void`                                                     | -                | Función llamada al seleccionar una opción            |
| `icon`             | `ReactNode`                                                                   | -                | Ícono a mostrar al inicio del dropdown               |
| `showSelectedIcon` | `boolean`                                                                     | `true`           | Muestra el ícono de selección en la opción elegida   |
| `defaultValue`     | `string`                                                                      | -                | Valor seleccionado por defecto                       |
| `value`            | `string`                                                                      | -                | Valor controlado del dropdown                        |
| `className`        | `string`                                                                      | -                | Clases adicionales para el contenedor                |
| `buttonClassName`  | `string`                                                                      | -                | Clases adicionales para el botón principal           |
| `itemClassName`    | `string`                                                                      | -                | Clases adicionales para los elementos de la lista    |
| `activeColor`      | `string`                                                                      | -                | Color personalizado para los elementos seleccionados |
| `checkColor`       | `string`                                                                      | -                | Color personalizado para el ícono de verificación    |

### DropdownItem

| Propiedad  | Tipo        | Descripción                     |
| ---------- | ----------- | ------------------------------- |
| `label`    | `ReactNode` | Contenido visible de la opción  |
| `value`    | `string`    | Valor de la opción              |
| `icon`     | `ReactNode` | Ícono opcional para la opción   |
| `disabled` | `boolean`   | Si la opción está deshabilitada |

## API Reference

| Prop             | Tipo                                                                                    | Por Defecto        | Descripción                                                                                |
| ---------------- | --------------------------------------------------------------------------------------- | ------------------ | ------------------------------------------------------------------------------------------ |
| items            | `DropdownItem[]`                                                                        | -                  | Lista de elementos a mostrar en el menú desplegable.                                       |
| placeholder      | `string`                                                                                | `"Seleccionar..."` | Texto a mostrar cuando no hay selección.                                                   |
| variant          | `"default"` \| `"primary"` \| `"secondary"` \| `"success"` \| `"warning"` \| `"danger"` | `"default"`        | Variante visual del botón desplegable.                                                     |
| size             | `"sm"` \| `"md"` \| `"lg"`                                                              | `"md"`             | Tamaño del componente.                                                                     |
| radius           | `"none"` \| `"sm"` \| `"md"` \| `"full"`                                                | `"md"`             | Radio de borde del componente.                                                             |
| disabled         | `boolean`                                                                               | `false`            | Deshabilita el componente.                                                                 |
| fullWidth        | `boolean`                                                                               | `false`            | Si es `true`, el componente tendrá ancho completo.                                         |
| placement        | `"bottom"` \| `"top"` \| `"left"` \| `"right"`                                          | `"bottom"`         | Posición donde aparece el menú desplegable.                                                |
| onChange         | `(value: string) => void`                                                               | -                  | Función a ejecutar cuando cambia el valor seleccionado.                                    |
| icon             | `React.ReactNode`                                                                       | -                  | Icono opcional para mostrar en el botón.                                                   |
| showSelectedIcon | `boolean`                                                                               | `true`             | Si es `true`, muestra un icono de verificación junto al elemento seleccionado.             |
| defaultValue     | `string`                                                                                | -                  | Valor seleccionado por defecto (no controlado).                                            |
| value            | `string`                                                                                | -                  | Valor seleccionado (controlado).                                                           |
| width            | `string`                                                                                | -                  | Ancho fijo para el menú desplegable.                                                       |
| activeColor      | `string`                                                                                | -                  | Color personalizado para los elementos seleccionados (ej: "#FF5500" o "rgb(100,150,200)"). |
| checkColor       | `string`                                                                                | -                  | Color personalizado para el icono de verificación (ej: "#FF5500" o "rgb(100,150,200)").    |

## Ejemplos

### Colores personalizados

Puedes personalizar el color de los elementos seleccionados y el color del ícono de verificación:

```tsx
<Dropdown
  items={[
    { label: "Default", value: "default" },
    { label: "Opción 2", value: "option2" },
    { label: "Opción 3", value: "option3", disabled: true },
  ]}
  activeColor="#FF5500"
  checkColor="#00AAFF"
/>
```

Por defecto, el ícono de verificación (✓) coincide con el color de la variante seleccionada:

- **default, primary**: azul
- **secondary**: púrpura
- **success**: verde
- **warning**: ámbar
- **danger**: rojo

Si deseas un color específico para el ícono, utiliza la prop `checkColor`.
