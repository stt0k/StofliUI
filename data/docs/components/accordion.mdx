---
title: "Accordion"
description: "Un componente de acordeón que permite mostrar y ocultar contenido de manera interactiva."
---

## Instalación

```bash
# Instalar dependencias necesarias
npm install framer-motion @heroicons/react

# Instalar el componente
npx @stofli/ui add accordion
```

## Uso

El componente Accordion acepta un array de items con título y contenido, y opcionalmente puede configurarse para permitir múltiples secciones abiertas.

```tsx
import { Accordion } from "@stofli/ui";

export default function App() {
  return (
    <Accordion
      items={[
        {
          title: "¿Qué es un Accordion?",
          content: "Un accordion es un componente que permite mostrar y ocultar contenido...",
        },
        {
          title: "¿Cuándo debería usarlo?",
          content: "Los accordions son útiles cuando necesitas presentar mucha información...",
        },
      ]}
    />
  );
}
```

## Props

| Prop          | Tipo                                          | Default     | Descripción                                    |
| ------------- | --------------------------------------------- | ----------- | ---------------------------------------------- |
| `type`        | `"single" \| "multiple"`                     | `"single"`  | Tipo de acordeón                              |
| `defaultValue`| `string \| string[]`                         | -           | Valor inicial del acordeón                     |
| `value`       | `string \| string[]`                         | -           | Valor controlado del acordeón                  |
| `onValueChange`| `(value: string \| string[]) => void`       | -           | Función llamada al cambiar el valor           |
| `className`   | `string`                                     | -           | Clases adicionales                             |
| `variant`     | `"default" \| "primary" \| "secondary" \| "success" \| "warning" \| "danger"` | `"default"` | Variante de estilo |
| `radius`      | `"none" \| "sm" \| "md" \| "full"`          | `"md"`      | Radio de las esquinas                         |
