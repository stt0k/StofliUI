    <HeadingAnchor id="tailwind-v3" level={3}>Instalación para Tailwind CSS v3</HeadingAnchor>

    Instala Tailwind CSS v3 y sus dependencias:

```bash
npm install -D tailwindcss@3 postcss autoprefixer
npx tailwindcss init -p
```

Configura tu archivo `tailwind.config.js` para Tailwind CSS v3:

```js
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/stofli-ui/dist/**/*.{js,ts,jsx,tsx}",
  ],
  darkMode: "class",
  theme: {
    extend: {},
  },
  plugins: [],
};
```

Crea o actualiza tu archivo CSS principal (por ejemplo, `src/index.css`) con las directivas de Tailwind v3:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

Y asegúrate de que tu `vite.config.js` tenga configurado el plugin de React:

```js
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
 
export default defineConfig({
  plugins: [
    react(),
  ],
  define: {
      'process.env': {}
    },
})
```

    <HeadingAnchor id="tailwind-v4" level={3}>Instalación para Tailwind CSS v4</HeadingAnchor>

    Para Tailwind CSS v4, el proceso es diferente. Instala Tailwind y el plugin de Vite específico:

```bash
npm install tailwindcss @tailwindcss/vite
```

Actualiza tu `vite.config.js` para incluir el plugin de Tailwind CSS v4:

```js
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";
import tailwindcss from "@tailwindcss/vite";

export default defineConfig({
  plugins: [react(), tailwindcss()],
  define: {
      'process.env': {}
    },
})
```

Y crea o actualiza tu archivo CSS principal (por ejemplo, `src/index.css`) con la nueva sintaxis de importación de Tailwind v4:

```css
@import "tailwindcss";
@source "../node_modules/stofli-ui/dist/**/*.{js,ts,jsx,tsx}";
```

Tienes dos opciones para trabajar con Tailwind CSS v4:

1. **Sin archivo de configuración**: El plugin `@tailwindcss/vite` gestiona la configuración automáticamente y detecta los archivos usando la directiva `@source` en tu CSS.

2. **Con archivo de configuración**: Puedes crear un archivo `tailwind.config.js` e importarlo en tu CSS:

```js
// tailwind.config.js (opcional en v4)
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
    "./node_modules/stofli-ui/dist/**/*.{js,ts,jsx,tsx}",
  ],
  darkMode: "class",
  theme: {
    extend: {},
  },
};
```

Si optas por usar el archivo de configuración, debes importarlo en tu CSS:

```css
@import "tailwindcss";
@config "../tailwind.config.js";
```

En este caso, no necesitas usar la directiva `@source` ya que la detección de clases se hará a través de la propiedad `content` del archivo de configuración.
